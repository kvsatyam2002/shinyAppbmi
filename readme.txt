This shiny application has been created as a part of a Data Science assignment on Coursera. 
After opening the application in R, please follow these instructions.
1.Enter personal details in the left panel and press enter.
2.On the Main panel the entered data is displayed.
3.The application calculates and diplays the BMI(Body Mass Index), based on the values entered under weight and height.
4.Thee is also a graphical representation of BMI trends wherein the calculated BMI is indicated by a blue line.
5.Therein,it grades the BMI calculated according as whether the entered data represents a 'Normal weight', 'Over weight' or 'Under weight' individual.
6.Changing the weight or height entries, changes the position of the blue line.
